# Multilayer Network Library

**Author:** Ondřej Sivek

## Description
Master's thesis project – a library in TypeScript/JavaScript for representing, analyzing, and visualizing multilayer networks.  
The library allows users to model networks with multiple interconnected layers, perform analysis with built-in algorithms, and create interactive visualizations in the browser.

## Features
- Represent nodes and links across multiple layers
- Perform network analysis (e.g., centrality, connectivity)
- Visualize networks interactively
- Modular and extendable architecture

## Technologies
- TypeScript/JavaScript
- HTML/CSS
- Web visualization library (AntV G6)
